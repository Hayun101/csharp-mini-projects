# csharp-mini-projects
This repository contains four small C# console applications
Each project focuses on a different core programming concept

Projects:

todo-app
A basic todo list application.
Demonstrates console input/output, collections, control flow, and basic program structure.

student-oop
A simple student management application.
Demonstrates object-oriented programming concepts such as classes, constructors, encapsulation, and object collections.

bank-system
A basic banking simulation.
Demonstrates business logic, multiple interacting classes, validation, and state management.

file-io
A simple file input/output example.
Demonstrates reading data from a file, writing data to a file, and maintaining data persistence between runs.
